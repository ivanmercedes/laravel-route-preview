# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] Stable Release - 2025-12-25

### Added

- Modern card-based UI with dark mode support
- Route grouping by URI prefix or controller class
- Export routes to JSON or CSV formats
- Statistics dashboard with route counts and method distribution
- Click controller/action to open file in VSCode
- Copy URI and actions to clipboard
- Expandable middleware sections for cleaner view
- Full accessibility with ARIA labels and keyboard navigation

### Changed

- Redesigned from table to modern card layout
- Improved header organization with compact 2-row layout
- Enhanced search functionality with clear button
- Better visual hierarchy and color-coded HTTP methods
